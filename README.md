# Baggage AI

```
1. cd to the project directory
2. activate your virtualenv
3. run: pip install -r requirements.txt
4. run: python backend.py
```
## Database

 ```
1. BaggageAI backend project runs on flask framework along with mongoDB noSQL Database 
2. MongoDB is a document based databased which provides lighting fast read-write architecture
3. Flask supports noSQL Database integration which makes mongoDB best suitable for its scenario
4. MongoDB has horizontal scaling feature along with data encryption in BSON(Binary script object notation) format
```
